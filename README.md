# Hotel Booking System

A comprehensive web application for hotel room booking built with Spring Boot and Maven.

## Features

- User Registration and Authentication
- Hotel and Room Management
- Room Availability Search
- Booking Management
- Payment Processing
- User Reviews and Ratings
- Admin Dashboard
- JWT-based Security

## Technology Stack

- **Backend**: Spring Boot 2.7.0, Spring Security, Spring Data JPA
- **Database**: MySQL
- **API Documentation**: Swagger/OpenAPI
- **Authentication**: JWT
- **Build Tool**: Maven
- **Java Version**: 11

## Getting Started

### Prerequisites

- Java 11 or higher
- Maven 3.6+
- MySQL 5.7+

### Installation

1. Clone the repository
2. Configure MySQL connection in `application.properties`
3. Run the application:

```bash
mvn clean install
mvn spring-boot:run
```
